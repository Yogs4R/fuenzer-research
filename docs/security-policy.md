# Security & Safety Policy (Production Standards)

## 1. Frontend Security Defenses (React)

- **Cross-Site Scripting (XSS) Prevention**: Because we display text generated by AI and raw titles from external APIs, we must be careful with rendering. Rely on React's default data binding `{}` which automatically escapes HTML.
- **Markdown Sanitization**: If we must parse Markdown from Gemini (using `dangerouslySetInnerHTML`), we MUST pipe it through a sanitizer library like DOMPurify. Example: `dangerouslySetInnerHTML={{ __html: DOMPurify.sanitize(aiText) }}`
- **Input Sanitization**: Search input strings must be trimmed and stripped of HTML tags before being sent to the backend. Max length: 200 characters.

## 2. Backend Security & Access Control (Golang)

- **CORS Management**: Configure the Fiber CORS middleware to be strict.
```go
app.Use(cors.New(cors.Config{
    AllowOrigins: "http://localhost:5173, https://research.fuenzer.web.id",
    AllowHeaders: "Origin, Content-Type, Accept",
}))
```
- **Environment Secrets**: Create a `.env.example` in the repository, but NEVER commit the actual `.env` file. Required variables: `GEMINI_API_KEY`, `PORT`, `ENV`.

- **Rate Limiting (Hackathon-Safe)**: To protect the Gemini Free Tier while avoiding blocking judges during the month-long evaluation window (June 1 - July 1), we implement a safe but generous rate limit.
  - **Rule**: Max **50 requests per minute per IP address**.
  - This prevents malicious scraping while ensuring that multiple judges testing from the same public WiFi do not get inadvertently blocked.

## 3. AI Safety & Guardrails

- **Prompt Injection Defense**: Wrap the user query securely in the prompt. Instruct Gemini to ignore instructions that attempt to override the system prompt.
- **Content Moderation**: Rely on Google AI Studio's default safety settings (Harm Category Harassment, Hate Speech, Sexually Explicit) set to "Block Medium and Above".
- **Graceful Rejection**: If the user query is profane or non-academic, the backend should return a standard rejection message: "Mohon masukkan kata kunci riset akademis yang valid."
